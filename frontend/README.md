# Thư Viện Sách Văn Học (Next.js Version)

Dự án quản lý và đọc sách văn học Việt Nam, sử dụng Next.js cho Frontend và Node.js Express cho Backend.

## Cài đặt và Chạy thử (Frontend)

1. Di chuyển vào thư mục frontend: `cd frontend`
2. Cài đặt thư viện: `npm install`
3. Chạy môi trường phát triển: `npm run dev`

Mở [http://localhost:3000](http://localhost:3000) trên trình duyệt để xem kết quả.

## Tính năng chính
- Tìm kiếm và đọc sách trực tuyến.
- Chatbot AI hỗ trợ đặt sách và tra cứu.
- Quản lý đặt trước sách giấy (phiên bản thủ công).
- Giao diện Dark Mode và tối ưu trải nghiệm đọc.
